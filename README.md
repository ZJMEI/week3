# Week 3

User input and domain modeling.

You do NOT have to fork this repository.

Clone this repository, `cd` into the application directory, then:

```
$ bundle install
```

Assuming all goes well (you should see a "Bundle complete!" message):

```
$ rails server
```


### Rails Commands

* `rails new`
* `rails server`
* `rails console`
* `rake routes`
* `rake db:migrate`
* `rake db:seed`
